# Dataset

The datasets used in this project are large geospatial raster files, so they are not stored directly in this GitHub repository.

You can download them from the following links:

## Files

Riyadh_LST.tif  
Land Surface Temperature data exported from Google Earth Engine.  
Download: (https://drive.google.com/file/d/1Ul5knWxq2qdZ_9Jdqkp7ImGOakIbzkv3/view?usp=sharing)

Riyadh_NDVI.tif  
Normalized Difference Vegetation Index derived from Landsat imagery.  
Download: (https://drive.google.com/file/d/1a6e7RuSYD17hl37V5fJ71b40GzHTjvew/view?usp=sharing)

Riyadh_Urban.tif  
Urban expansion / built-up areas dataset.  
Download: (https://drive.google.com/file/d/1jyEJKY8Oy90PjgbRUneXk7UhSxXPyDFz/view?usp=sharing)

## Data Source

The datasets were generated using:

- Google Earth Engine
- Landsat 8 Surface Temperature dataset
- NDVI calculated from Landsat bands
- ESA WorldCover / Built-up dataset

Study Area: **Riyadh, Saudi Arabia**
